This script is a demo to authenticate PowerBI API and create example dataset on specified workspace

#Environment Installation:
    -Install Python (>=3.7.4)
    -Move root directory and activate virtural environment by running the following command in terminal
        ```source ./pbienv/bin/activate``` For macOS
        ```./pbienv/bin/activate``` For windows

#Running Script:
    -Confirm config.json file to check if it's properly configed 
    -Run script by running the following commnad in terminal
        ```python demo.py config.json```
